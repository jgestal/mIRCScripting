Scytale ha muerto, viva CHaN!

    Tutorial sobre los bots del IRC-Hispano.org - Por Hiperman

    CHaN permite registrar y controlar varios aspectos de los canales. CHaN ayuda a la administración de los canales permitiendo establecer ciertos límites según el nivel del usuario. Cualquier usuario con un nick registrado puede registrar un canal con el comando REGISTER. Si ningún registrado entra en 15 dís el canal será borrado.

   Para que CHaN ejecute una acción, tienes que enviarle un mensaje (o estar en un query) en el que ponga el comando apropiado. Por ejemplo: /msg chan help (o escribir "help" en el query con CHaN).

   A continuación estan todos los comandos. La sintaxis a seguir es: /msg CHaN COMANDO [parámetros obligatorios] (parámetros optativos)

 

HELP (comando) (opción) Da ayuda sobre el comando especificado o sobre la opción de este si se ha especificado. Si no se especifica ninguno de los dos se da ayuda general sobre CHaN.

REGISTER [canal] [contraseña] [descripción]

Registras el canal especificado. Para registrarlo tienes que ser operador en el canal y tu nick debe estar registrado. La contraseña especificada será usada para que te identifiques con el comando IDENTIFY como el FOUNDER (fundador) del canal.
DROP [canal] Elimina el canal especificado. Este comando solo puede ser utilizado por el FOUNDER, quien debe estar identificado.

IDENTIFY [canal] [contraseña]

Te identificas como el FOUNDER del canal especificado. Identificarte como fundador del canal es necesario para poder ejecutar varios comandos.

SET [canal] FOUNDER [nick]

Cambia el fundador del canal por el nick especificado.

SET [canal] PASSWORD [contraseña]

Cambia la contraseña que utiliza el fundador para identificarse por la contraseña especificada.

SET [canal] DESC [descripción]

Fija la descripción del canal. Esta será mostrada con los comandos INFO Y LIST.

SET [canal] URL (url)

Asocia la url especificada al canal. Esta será mostrada con el comando INFO. Si no se especifica ninguna url la url asociada al canal será borrada.

SET [canal] EMAIL (mail)

Asocia la dirección de correo especificada al canal. Esta será mostrada con el comando INFO. Si no se especifica ningun e-mail el e-mail asociado al canal será borrado.

SET [canal] ENTRYMSG (mensaje)

Especifica el mensaje que CHaN enviará via NOTICE a los usuarios cuando entren al canal. Si no se especifica ningún mensaje no habrá mensaje de bienvenida.

SET [canal] TOPIC [texto]

Cambia el TOPIC (tema del canal) por el texto especificado.

SET [canal] KEEPTOPIC [on|off]

On: activa la retención de topic. Cuando todos los usuarios abandonen el canal CHaN recordará el topic actual y lo establecerá la próxima vez que entre un usuario en el canal.
Off: desactiva la retención de topic.

SET [canal] TOPICLOCK [on|off]

On: activa la traba de topic. Solo será posible cambiar el topic mediante el comando SET TOPIC.
Off: desactiva la traba de topic.

SET [canal] MLOCK [modos]

Traba los modos especificados. Los modos pueden ser trabados para que no se puedan activar (con "-") o para que no se puedan desactivar (con "+"). Si en el parámetro de los modos solo se especifica "+" todos los modos podrán ser activados y desactivados. Por ejemplo con "SET #100scripts MLOCK +nt-spk" no se podrán quitar los modos n y t, no se podrán poner los modos s, p y k y los demas modos se podrán poner o quitar libremente.

SET [canal] RESTRICTED [on|off]

On: activa el acceso restringido. Los usuarios con acceso negativo que entren al canal serán kickbaneados (serán expulsados y se le impedirá el paso al canal). Si el modo SECUREOPS está activado solo podrán permanecer en el canal los usuarios registrados.
Off: desactiva el acceso restringido.

SET [canal] SECURE [on|off]

On: activa la seguridad del canal. Solo podrán permanecer en el canal los usuarios registrados. Los usuarios no registrados serán kickbaneados.
Off: desactiva la seguridad del canal.

SET [canal] SECUREOPS [on|off]

On: activa la seguridad de operadores. Solo los usuarios registrados pueden obtener op (estatus de operador).
Off: desactiva la seguridad de operadores.

ACCESS [canal] ADD [nick] [nivel]

Registra (añade a la lista de usuarios del canal) al nick especificado con el nivel de accesoespecificado. Si el usuario está actualmente registrado solo se modificará su nivel. El usuario que utiliza este comando debe tener un nivel superior al nivel anterior del usuario al que se va a modificar (0 si no estaba registrado)
ACCESS [canal] DEL [nick | número] Elimina de la lista de usuarios del canal al usuario especificado (mediante su nick o su número en la lista). Para ejecutar este comando es necesario tener un nivel superior al nivel del usuario.

ACCESS [canal] LIST (patrón)

Muestra la lista entera de los registrados del canal. Se pueden especificar nicks con carácteres comodines o máscaras en cuyo caso solo se listarán las coincidencias.
HELP LEVELS DESC Lista todo los comandos especificables.
LEVELS [canal] SET [comando] [nivel] Establece el nivel mínimo necesario para ejecutar el comando especificado. Los comandos SET FOUNDER y SET PASSWORD solo pueden ser modificados por el founder.
LEVELS [canal] DIS | DISABLE [comando] Solo el founder podrá ejecutar el comando especificado. El comando DISABLE se puede ejecutar tambien con DIS (la forma abreviada de DISABLE).
LEVELS [canal] LIST Muestra el nivel actual necesitado para ejecutar cada comando.
LEVELS [canal] RESET Establece todos los comandos con el nivel por defecto, esto es, los niveles pre-establecidos para cuando se registra un canal.
AKICK [canal] ADD [mask] (razón) Agrega la mask especificada a la lista de expulsiones automáticas. Cuando este usuario entre en el canal CHaN lo kicbaneará con el mensaje especificado en "razón". Si este no se ha especificado el mensaje utilizado será "You have been banned from the channel".
AKICK [canal] DEL [mask] Elimina la mask especificada de la lista de expulsiones automáticas.
AKICK [canal] LIST (mask) Muestra la lista de expulsiones automáticas. Si se especifica "mask" solo se listarán las expulsiones automáticas que coincidan con esta mask (se pueden utilizar carácteres comodínes).
INFO [canal] Muestra información sobre el canal o te avisa si este no está registrado. Lista el fundador, cuando se registró, cuando fue la última vez que un usuario registrado entró, la descripción del canal y los modos trabados del canal.
LIST [patrón] Lista todos los canales que coincidan con el patrón especificado, a menos que tengan la opción PRIVATE activada.

OP [canal] [nick]

Hace operador al nick especificado. Por defecto se necesita nivel 5.

DEOP [canal] [nick]

Deopea (quita el estado de operador) al nick especificado. Por defecto se necesita nivel 5.

UNBAN [canal]

Elimina todos los bans del canal. Por defecto se necesita nivel superior a 0.

INVITE [canal]

Se invita a si mismo al canal. Por defecto se necesita nivel superior a 0.

CLEAR [canal] [modes]

Quita los modos establecidos en el canal. Por defecto se necesita un nivel de 10.

CLEAR [canal] [bans]

Elimina todos los bans del canal. Por defecto se necesita un nivel de 10.

CLEAR [canal] [ops]

Deopea a todos los operadores del canal. Por defecto se necesita un nivel de 10.

CLEAR [canal] [voices]

Quita la voz a todos los usuarios que la tuviesen. Por defecto se necesita un nivel de 10.

CLEAR [canal] [users]

Kickea a todos los usuarios del canal. Por defecto se necesita un nivel de 10.
