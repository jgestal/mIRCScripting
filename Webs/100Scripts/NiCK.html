    Tutorial sobre los bots del IRC-Hispano.org - Por Hiperman

    NiCK te permite registrar un nick y previene que otros usuarios hagan uso de él. Tener un nick registrado es obligatorio para poder utilizar a MeMO o estar registrado en un canal. Si un nick no es utilizado durante 15 días sera borrado. Para registrar un nick usa el comando REGISTER.

   Para que NiCK ejecute una acción, tienes que enviarle un mensaje (o estar en un query) en el que ponga el comando apropiado. Por ejemplo: /msg nick help (o escribir "help" en el query con NiCK).

   A continuación estan todos los comandos. La sintaxis a seguir es: /msg NiCK@deep.space COMANDO [parámetros obligatorios] (parámetros optativos)

 

REGISTER [e-mail]

Esto es lo primero que deberías hacer. Con este comando se registrará el nick que estás usando y el password se enviará al e-mail que hayas puesto. No son válidas ni cuentas de correo web ni anonimizadores.

DROP

Eliminas tu nick de la lista de nicks registrados de NiCK. Una vez hecho esto cualquier persona puede registrar este nick.

IDENTIFY [password]

Te identificas como el propietario del nick que estás usando. Necesitas estar autentificado para ejecutar la mayoria de los comandos que siguen.

ACCESS LIST

Muestra la lista de máscaras que NiCK aceptará como validas a la hora de intentar identificarte.

ACCESS ADD [máscara]

Añade una máscara a la lista de máscaras validas para identificarte.

ACCESS DEL [máscara]

Quita una máscara a la lista de máscaras validas para identificarte.

LINK [nick] [contraseña]

Une tu nick con el nick que has especificado. Cuando te identifiques con uno de los dos nicks automáticamente te identificarás con el otro y todos los mensajes enviados a cualquiera de los dos nicks serán almacenados en la misma lista y se podrán leer desde cualquiera de los dos nicks. Ademáas, si uno de los dos nicks está registrado en un canal el otro nick tendrá los mismos privilegios que este.

UNLINK [nick]

Separa dos nicks que anteriormente estaban unidos. Ya no se compartirán ni los mensajes ni los registros. Los mensajes se quedarán en el nick anterior.

SET PASSWORD [password]

Cambia la password antigua de tu nick por la que hayas especificado.

SET LANGUAGE [1-7]

Cambia el idioma con el que te responderan los bots de servicio. Estos son los idiomas aceptados:
1 - Inglés
2 - Japonés/EUC encoding
3 - Japonés/JIS encoding
4 - Japonés/SJIS encoding
5 - Portugués
6 - Castellano
7 - Turco

SET URL [url]

Asocia la url que hayas especificado con tu nick. Esta será mostrada con el comando INFO.

SET EMAIL [e-mail]

Asocia el e-mail que hayas especificado con tu nick. Este será mostrado con el comando INFO.

SET KILL [on|quick|immed|off]

On: activa la protección de kill. Si un usuario se pone tu nick deberá cambiárselo ántes de un minuto o será desconectado. Si la dirección del usuario aparece en las listas de acceso no será desconectado.
Quick: el usuario dispondrá de 20 segundos para cambiarse de nick en vez de 60.
Immed: el usuario será desconectado inmediatamente y sin previo aviso. Por favor, no utilizar si no es totalmente necesario.
Off: desactiva la protección de kill.

SET SECURE [on | off]

On: activa las funciones de seguridad. Es necesario enviar tu contraseña para identificarse, aun cuando tu dirección este en la lista de acceso.
Off: desactiva las funciones de seguridad.

SET PRIVATE [on | off]

On: activa la privacidad sobre tu nick. Esto evita que tu nick aparezca cuando alguien utilice el comando LIST.
Off: desactiva la privacidad sobre tu nick.

SET HIDE [email|usermask|quit] [on|off]

Esconde cierta información sobre tu nick al utilizar el comando INFO. "On" esconde el primer parámetro especificado y "Off" lo muestra.
Email: tu e-mail
Usermask: la última máscara
Quit: el último mensaje de sálida

RECOVER [nick] (contraseña)

Se conecta un usuario falso con el nick que has especificado haciendo que el portador de este nick sea killeado. El usuario falso permanecera conectado un minuto. Para utilizar este comando tu dirección actual en el IRC tiene que ser una de las máscaras válidas del nick o tienes que proporcionar la contraseña del nick.

RELEASE [nick] (contraseña)

El usuario falso conectado con el comando RECOVER se desconectará inmediatamente dejando el nick libre. Para utilizar este comando tu dirección actual en el IRC tiene que ser una de las máscaras válidas del nick o tienes que proporcionar la contraseña del nick.

GHOST [nick] (contraseña)

Elimina una sesión fantasma del IRC. Para utilizar este comando tu dirección actual en el IRC tiene que ser una de las máscaras válidas del nick o tienes que proporcionar la contraseña del nick.

INFO [nick]

Muestra información relacionada con el nick que has especificado.

LIST [patrón]

Lista los nicks que concuerden con el patrón que hayas especificado, con el formato nick!user@host y aceptando comodines. Si un nick tiene la opción PRIVATE activada no se listará.

STATUS [nick1] (nick2) (...)

Informa sobre el estado del nick. La respuesta tiene el formato NICK 0-3
0 - El dueño no esta conectado o el nick no está registrado
1 - El portador del nick no es su dueño
2 - El portador del nick es reconocido por su dirección
3 - El portador del nick es su dueño
